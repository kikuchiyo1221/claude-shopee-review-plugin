# Shopee Product Listing Reviewer

Shopeeタイ向け商品タイトル・説明文のドラフトをレビューし、タイ語のネイティブ表現、業界専門用語、SEO最適化の観点からフィードバックを行い、完成版を生成する。

## When To Use

- ドラフトの商品タイトルや説明文のレビューを依頼された時
- 「この商品説明をチェックして」「レビューして」
- 「AパターンとBパターンどちらがいいか」と比較を求められた時
- 「タイ語が自然か確認して」「SEO的にどうか」
- Shopee出品前の最終チェック

## レビュー観点（3軸）

### 1. タイ語ネイティブ表現チェック

タイ人の購買者視点で不自然な表現を検出する。

**チェック項目:**

| 観点 | 具体例 |
|------|--------|
| 直訳調の表現 | 機械翻訳的な語順・助詞の誤用 |
| 口語 vs 書き言葉 | Shopeeでは適度にカジュアルが好まれる |
| 冗長な表現 | タイ語は簡潔さが重視される |
| 敬語レベル | ค่ะ/ครับ の適切な使用、過剰な敬語は不自然 |
| 文末表現 | นะคะ/นะครับ など親しみやすさの調整 |
| スペース・句読点 | タイ語のスペース規則（単語間不要、フレーズ間で使用） |

**よくある不自然表現パターン:**

```
❌ สินค้านี้มีคุณภาพที่ดีมาก（直訳調、ที่ が不要）
✅ สินค้าคุณภาพดีมาก

❌ ผลิตภัณฑ์ของเรา（商品説明で「弊社の製品」は堅すぎる）
✅ สินค้าตัวนี้ / สินค้าชิ้นนี้

❌ กรุณาสั่งซื้อ（丁寧すぎる）
✅ สั่งเลย / สั่งได้เลย

❌ ถ้าคุณต้องการ（直訳調の「もしあなたが欲しいなら」）
✅ ใครที่กำลังมองหา / สำหรับคนที่ต้องการ
```

### 2. 業界専門用語チェック

商品カテゴリごとに適切な専門用語が使われているか確認する。

**汎用チェック項目:**

| 観点 | 内容 |
|------|------|
| 正式名称 | 業界で一般的に使われる用語か |
| タイ語 vs 英語 | その業界でどちらが主流か（例: 美容は英語多め） |
| 単位・規格 | タイで一般的な単位表記か |
| ブランド名表記 | 公式の表記と一致しているか |

**カテゴリ別の注意点:**

- **園芸用品**: กรรไกรตัดกิ่ง（剪定鋏）、ใบมีด（刃）、ด้ามจับ（ハンドル）など正確な用語を使用
- **ホビー・玩具**: ของแท้ 100%（正規品100%）、ลิขสิทธิ์แท้（正規ライセンス）は必須
- **美容・コスメ**: 英語ベースの用語が主流（Serum, Essence, Moisturizer）、タイ語併記が効果的
- **電子機器**: スペック表記は英語＋タイ語説明
- **食品**: อย.（FDA認証）表記、原材料のタイ語表記
- **ファッション**: サイズ表記はタイ規格に合わせる

### 3. SEOチェック

Shopeeタイの検索アルゴリズムに最適化されているか確認する。

**タイトル（100文字以内推奨）:**

| 要素 | 優先順位 | 例 |
|------|----------|-----|
| ブランド名 | 1 | [OKATSUNE] |
| 商品名（タイ語） | 2 | กรรไกรตัดกิ่ง |
| 型番・サイズ | 3 | รุ่น 103 |
| 主要特徴 | 4 | คมกริบ ทนทาน |
| 差別化キーワード | 5 | ผลิตในญี่ปุ่น ของแท้ 100% |

**タイトルのSEOルール:**
- キーワードは前方に配置（検索ウェイトが高い）
- キーワードの羅列は避ける（スパム判定リスク）
- 括弧で区切って可読性を確保
- 同義語は1つだけ選ぶ（重複は逆効果）

**説明文のSEO:**
- 冒頭100文字に主要キーワードを含める
- 自然な文脈でキーワードを散りばめる
- ハッシュタグは5-10個（多すぎると逆効果）
- 構造化（見出し・箇条書き）で検索スニペット対応

## レビュー時の注意事項

- ドラフトの意図を尊重し、不必要な変更は避ける
- 修正は必ず理由を添える（「タイ人には不自然」だけでなく具体的に説明）
- 完成版は必ずタイ語＋日本語訳のセットで出力する
- SEOキーワードの提案は商品カテゴリに応じて調整する
- 誇大広告（เกินจริง）にならないよう注意する
